# project-core

Personal Discord bot. Talks like a real person.
Built with Discord.js v14 and OpenRouter API.

## Setup
1. Clone the repo
2. npm install
3. Fill in .env — see .exnv for reference
4. npm run deploy
5. npm start

## Stack
- Discord.js v14
- OpenRouter API
- Node.js 18+
- JSON file persistence
- Railway for hosting

## Notes
Model is set in conf/aicf.json — change it there.
Personality files live in mind/ — edit those to
change how the bot talks and what it knows.
